Service FAQ

How to Register a Special/Forming tool?
  • IN 2D CAD OPEN A NEW FILE
  • MAKE THE DRAWING OF THE SPECIAL TOOL AND THEN CLICK ON FILE
    • REGISTER SP/FORMING(E)
    • SP(T)
  • SELECT THE WHOLE COMPONENT
    • RIGHT CLICK COMPLETE
    • SELECT THE TOOLING CENTRE POINT
    • SELECT THE SIZE OF GUIDE (STATION)
    • GIVE THE MANAGEMENT NAME
    • SAVE
  • GO TO HOLE
    • SP(P)
    • AND LOAD THE SPECIAL SHAPE IN THE DRAWING.
How to take data backup?
  • GO TO DATAMANAGER
    • SELECT FOLDER NAME “*”
    • FIND
  • THEN SELECT ALL
    • REGISTER
  • IMPORT/EXPORT
    • FD OUTPUT
  • SELECT THE DRIVE IN WHICH YOU WANT TO TAKE DATA BACKUP.
  • EXECUTE
